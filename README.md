# technical-writing-argparse-documentation
Portfolio project that structurally rewrites the official Python documentation for the `argparse` module, to reduce user friction and onboarding time for new developers. 
## Project: Argparse Documentation Audit
### Objective: 
Transforming expert-level documentation into user-centric guides.

### Overview
This project demonstrates the ability to translate complex technical specifications into actionable, beginner-friendly instructions. I audited the official Python **argparse** tutorial to identify areas where information overload hindered user success.

### Key Improvements
- **Standardized Verbs:** Replaced passive voice with active, command-oriented language.
- **Logic Sequencing:** Reorganized steps to follow the natural developer workflow.
- **Visual Scannability:** Integrated bolded UI elements and consistent code-block formatting.

### Tools Used
- Markdown:  for structure.
- Google Developer Documentation Style Guide : for tone and grammar.
- GitHub: for version control.
